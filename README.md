# 무한의 계단 — 클래식(무한) 모드 웹 복원

Infinite Staircase classic/endless mode, rebuilt for the web from the decompiled
original (Unity IL2CPP). All gameplay constants (gauge drain/refill, speed ramp,
stair patterns, controls, effects) were recovered from `libil2cpp.so`; sprites,
backgrounds, gauge UI, buttons and audio were extracted from the original Unity bundle.

**Play:** open `index.html` (or the GitHub Pages URL).

- 올라가기: ↑ / 화면 오른쪽 / 오르기 버튼
- 방향바꾸기: Space / 화면 왼쪽 / 방향전환 버튼
- 모바일 터치 지원. 캐릭터 12종 선택.
